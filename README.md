This repo is an example of using kafka to log streaming data

![](asset/system_architect.png)

## Installation

```
docker compose up -d
```

After all apps are run

- zookeeper: http://localhost:9021/
- airflow : http://localhost:8080/

## Technologies

- Apache Airflow
- Apache Kafka
- Apache Zookeeper
- Apache Spark
- Cassandra
- PostgreSQL
- Docker
